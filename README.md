# Chatbot Slot Validation Error Handling

This example demonstrates a simple slot-filling chatbot in Python. It collects user information (name, email, age) through a conversational flow. The core concept illustrated is how to gracefully handle data validation errors, providing user-friendly feedback without exposing internal system details, and re-prompting for invalid slots.

## Language

`python`

## How to Run

1. Save the code as `main.py`.
2. Run from your terminal: `python main.py`
